# E-Commerce Data Analysis using SQL & Python

##  Project Overview
This project analyzes an E-Commerce dataset using SQL and Python in Jupyter Notebook.  
The goal of this project is to extract meaningful business insights from sales data.

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib / Seaborn
- SQL
- Jupyter Notebook

## Key Analysis Performed
- Top Selling Products Analysis
- Monthly Revenue Trends
- Customer Purchase Behavior
- Region-wise Sales Performance
- Order Status Analysis

## Key Insights
- Identified top revenue generating categories
- Observed monthly sales growth patterns
- Analyzed customer buying trends
- Found high-performing regions

## Project Structure
- `data/` → Dataset used
- `notebook.ipynb` → Jupyter Notebook with SQL & Python analysis

## How to Run the Project
1. Clone the repository
2. Open the Jupyter Notebook
3. Run all cells
